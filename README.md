# 🔐 Image Encryption Tool (Python)

A Python-based image encryption tool that uses pixel-level manipulation and XOR operations to securely encrypt and decrypt images. This project was developed as part of my cybersecurity internship to understand how data can be transformed at a binary level.

---

## 🚀 Features

- Encrypt images using pixel-wise XOR transformation
- Decrypt images using the same key
- Supports RGB and RGBA images
- Uses NumPy for efficient image processing
- Generates strong encryption using a random key matrix
- Preserves image integrity using lossless formats (PNG)

---

## 🧠 Concepts Covered

- Image Processing (Pixel Manipulation)
- XOR Encryption Technique
- Symmetric Encryption
- NumPy Array Operations
- Lossy vs Lossless Compression (JPG vs PNG)

---

## 🛠️ Tech Stack

- Python 3
- Pillow (PIL)
- NumPy
